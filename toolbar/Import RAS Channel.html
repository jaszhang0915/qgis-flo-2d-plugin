Import HEC-RAS Channel¶

  1. Click the Import RAS Geometry icon to import channel data from a GeoRAS project.

  2. The tool will import the project from the *.g0 geometry

  3. For FLO-2D, Limit the cross sections to the bank stations.

  1. The imported RAS channel is saved to the User Layers.

  2. The user can edit the channel placement or cross sections so the channel will be schematized correctly.

  3. See the Schematize Channel Editor in the FLO-2D Widgets section below for further instructions.
